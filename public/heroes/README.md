# 영웅 초상화 이미지

이 폴더에 영웅 번호로 된 PNG 파일을 넣으면 사이트에 자동으로 초상화가 표시됩니다.

- 파일명 규칙: `{heroNo}.png` (예: `12.png`, `103.png`)
- 원본 도구(`ProjectC.exe`)의 `imgs/` 폴더에 있던 그 파일들과 동일합니다.
  (onedir 빌드의 `_internal/imgs/` 위치 — `.exe` 단독 파일에는 포함돼 있지 않음)
- 이미지가 없는 영웅은 번호로 자동 폴백됩니다.

이 폴더는 GitHub Pages 빌드 시 사이트 루트의 `heroes/` 로 복사됩니다.
